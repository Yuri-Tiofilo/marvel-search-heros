Marvel Heros Search

## Como rodar o projeto:

### Instalar as dependencias
`````js
yarn install
`````

### Rodar a aplicação
`````js
yarn start
`````

### Commit

`````js
yarn commit
`````

### Run Build

`````js
yarn build
or
git push origin main // CI/CD Netlify
`````
